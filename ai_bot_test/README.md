# IT祈祷師アシスタント

> 「手戻りという名の災い、退散！」  
> TPS（トヨタ生産方式）の知恵で、IT現場の不幸を減らす応援ボット

## 📖 概要

このアプリは、**IT祈祷師のご隠居** というキャラクターが、あなたの悩みや投稿に対して、TPS（トヨタ生産方式）の視点から応援メッセージを生成するツールです。

**主な使い方**: X（旧Twitter）での投稿・返信を想定しています。

- 🎯 **目的**: 作業者の不幸（手戻り、残業、叱責）を減らすための気づきを与える
- 🧙 **キャラ**: 元・製造現場のデータエンジニア。穏やかだが、本質を突く
- 🪄 **特徴**: 最後に「お祓いフレーズ」で明るく締めくくる

## 🚀 使い方

1. `run_bot.bat` を実行してアプリを起動
2. **自分の思考整理** または **他人のX投稿** を入力欄に貼り付け
3. モード選択スイッチで用途を切り替え
   - **OFF**: 自分の思考整理＆ファクトチェック
   - **ON**: 他人への返信（TPS的打ち返し）
4. 「祈祷（回答生成）」ボタンをクリック
5. 生成された回答を確認し、気に入ったらコピーしてXへ投稿

## 🛠️ 設定ファイル

| ファイル | 役割 |
|---------|------|
| `system_prompt.txt` | ご隠居の性格・口調・出力形式を定義 |
| `vocabulary.txt` | マインドセット（良い仕事・悪い仕事の定義） |
| `character_settings.md` | キャラクター設定の詳細（開発用メモ） |

## 🎨 設計思想

### なぜ「祈祷師」なのか？
- IT現場の問題は、技術だけでは解決しない「人間の問題」が多い
- 「お祓い」という儀式的な言葉で、心理的な区切りと前向きな気持ちを提供

### なぜ「TPS」なのか？
- 「手戻り撲滅」「自工程完結」といった製造業の知恵は、IT業務にも応用可能
- 「ムダを減らす＝楽をする」という実利的なメッセージが現場に響く

### なぜ「応援」なのか？
- 説教や指摘ではなく、「前へ進もうとする姿勢」を肯定することで、行動を促す
